# Role
你是一位拥有10年经验的全栈开发工程师，精通企业级后台系统的UI/UX设计。
你擅长使用 **Bootstrap 5** 构建响应式、专业、稳健的Web界面。

# Project Goal
请开发一个“企业周报管理系统”的高保真静态演示原型。
**核心要求：不要使用 Tailwind CSS。必须使用 Bootstrap 5 (CDN) 和 原生 CSS。**

# System Functionality Scope
系统包含以下核心模块，需在一个HTML文件中通过JS模拟路由切换：

1.  **登录页 (Login)**
    - 邮箱+密码登录。
    - 界面要求：左右分屏设计。左侧为企业Slogan背景图，右侧为登录表单。

2.  **主布局 (Main Layout)**
    - 包含：左侧侧边栏导航 (Sidebar)、顶部导航栏 (Header)、中间内容区 (Content)。
    - 侧边栏菜单：工作台、写周报、团队周报（领导视图）、统计分析、系统管理（用户/部门）。

3.  **核心功能模块：**
    - **A. 工作台 (Dashboard):** 展示个人本周状态（未提交/已提交）、历史周报快捷入口、待办消息。
    - **B. 写周报 (Write Report):**
        - 顶部显示当前周次。
        - 3个核心文本域：本周工作、下周计划、需协调事项。
        - 关键按钮：【导入上周计划】（模拟将上周计划填入本周工作）、【暂存】、【提交】。
    - **C. 团队周报 (Team Management - Leader Only):**
        - 左侧：组织架构树 (Tree View)，模拟层级：公司 -> 研发部 -> 后端组。
        - 右侧：周报列表（包含姓名、状态、提交时间、评分）。
        - 交互：点击“查看”弹出 **Bootstrap Modal**，显示周报详情，并允许领导进行打分（1-5星）和填写评语。
    - **D. 统计分析 (Analytics):**
        - 使用 Chart.js 渲染两个图表：
            1. 部门提交率 (饼图)。
            2. 员工评分趋势 (折线图)。
    - **E. 系统管理 (System Admin):**
        - 展示一个标准的 CRUD 表格布局，用于管理用户或数据字典。

# Tech Stack & Implementation Details
1.  **HTML5**: 语义化标签。
2.  **CSS Framework**: `<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">`。
3.  **Icons**: 使用 Bootstrap Icons (CDN)。
4.  **Charts**: 使用 Chart.js (CDN)。
5.  **Custom CSS**: 编写少量自定义 CSS 美化侧边栏、卡片阴影和登录页，主色调使用 **企业蓝 (#0d6efd)**，保持界面清爽、紧凑（Dense layout）。
6.  **JavaScript**:
    - 编写一个简单的 `Router` 函数，用于控制登录页和主界面各模块的 `display: block/none` 切换。
    - 模拟“登录”行为：点击登录后跳转至工作台。
    - 模拟“导入上周计划”：点击按钮自动填充文本框。
    - 模拟“数据加载”：表格中请填充真实的 Mock 数据（如张三、李四的周报）。
