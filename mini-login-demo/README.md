# 迷你登录系统

只有「登录 + 登录后显示一段文字」的极简示例，用于从零理解前后端如何配合。

## 怎么跑

```bash
# 1. 进入目录
cd mini-login-demo

# 2. 安装依赖（只需一次）
npm install

# 3. 启动
node server.js
```

浏览器打开 http://localhost:3000 ，用户名 `admin` 密码 `123456` 登录即可。

## 从零自己搭一遍
